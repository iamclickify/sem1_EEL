# 🌍 Planet Sorting Program in C

## 📘 Overview
This C program allows the user to **store information about planets** (name and diameter) and then **sort them by their diameter** in either ascending or descending order.  
It demonstrates the use of **arrays, structures (`struct`), user input, and bubble sort** — making it an excellent beginner-level project in C programming.

---

## 🚀 Features
- Allows manual input of planet names and diameters  
- Sorts planets in ascending or descending order  
- Displays the sorted list neatly in tabular format  
- Demonstrates practical use of `struct` in C  
- Beginner-friendly and well-commented code  

---

## 🧠 Concepts Used
- **Structures (`struct`)** — to store multiple properties (name + diameter) of a planet together  
- **Arrays** — to store multiple planets  
- **Loops** — for input and sorting  
- **Conditionals (if-else)** — to handle sorting order  
- **Bubble Sort Algorithm** — to rearrange planets by diameter  

---

## 🖥️ How It Works
1. The user enters the number of planets.  
2. The program asks for each planet’s **name** and **diameter**.  
3. The user chooses whether to sort in **ascending** or **descending** order.  
4. The program sorts the data using **Bubble Sort**.  
5. The final list of planets is displayed in a clean table format.
