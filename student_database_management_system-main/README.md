Student Database Management System in C
Description

This project is a simple console-based Student Database Management System written in C.
It demonstrates the use of structures, arrays, loops, and basic algorithms such as bubble sort and linear search.
The program allows users to add, sort, and search student records interactively through a menu-driven interface.
It is designed for beginners to understand the fundamentals of data handling and control structures in C programming.

Features

Add new student records (Name, Roll Number, PRN)

Sort students by PRN in ascending or descending order

Search for a student using PRN

Display student details after sorting or searching

Exit the program safely
